1.- Crear custom module 'Arrow for Home Page' en la región 'Slider/Hero' para la flecha y colocar el código: 

<div class="arrow-home"><a class="prev-icon arrow-bottom" href="http://google.com">link</a></div>


Tile <none> clase container-row only display in <front>


2.- Instalar Instagram social feed y configurarlo.

Colocar "Instagram Social Feed block" en la región Social Column 1

3.- Agregar un Custom block en blanco en "Social Column 4" con un espacio en blanco en body.

Tile <none> clase container-row only display in <front>

4.- Agregar un Custom block "Instagram Clone 1" en "Socila Column 7" con el siguiente codigo:

<script>
window.onload = function() {
     var x = document.getElementsByClassName("social-feed-item-2")[0];
     document.getElementsByClassName('region-social-col4')[0].appendChild(x);
     var x = document.getElementsByClassName("social-feed-item-3")[0];
     document.getElementsByClassName('region-social-col7')[0].appendChild(x);
};
</script>


Tile <none> clase container-row only display in <front>
